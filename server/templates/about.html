About This App

This app allows for easy creation of graphs and allows you to run different algorithms over them. I chose to do this becase I don't normally do web apps or GUIs and decided this was a good chance to tinker with it. The idea behind this is that it can be optimized to improve performance, more algorithms can be added and it can then be used for solving any problem that can be reduced to a graphing problem. With the way its set up, other applications could in theory turn their problem into a graph and request processing here. An example might be using a max flow algorithm to try and estimate the most efficient use of resources in a production line. Before using, PLEASE read the section below called "Algorithms and How to Use Them" that Gives an outline of all the currently implemented algorithms and how to use them in the graph.

Instructions

Vertices
- To place a vertex, click the "Vertex" button to enter vertex mode. Once in vertex mode you can click anywhere on the screen to add vertices.
Directed Edges
- Click the "Directed Edges" button to enter directed edges mode. In this mode you can continuously click on an origin vertex, and a destination vertex to create edges.
Undirected Edges
- Similar to directed edges mode except with undirected edges instead.
Edge Weights
- The weight text box indicates the weight of any edges to be placed. Click on the box and change the number to change the edge weight. If all edges are the same weight, it is essentially an unweighted graph.
Reset
- Clicking reset will remove all vertices and edges and start a new graph
Algorithms
- Click on the algorithms button for a dropdown list of available algorithms. Select the one you wish to run and press execute.

Algorithms and How to use Them

These must all have at least 2 vertices to work

Prims Algorithm
Finds a smallest spanning tree. Graph must be positive, undirected, connected and contain more than 1 vertex.
Breadth First Search (BFS)
Finds a spanning tree. Graph must be positive, connected and contain more than 1 vertex.
Depth First Search (DFS)
Finds a spanning tree. Graph must be positive, connected and contain more than 1 vertex.
